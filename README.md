## 🌱 About Me

> *"Technology is not just a tool — it's a canvas for building the future."*

I'm a passionate and ever-growing **Full-Stack Developer** who started my technology journey in **Rekayasa Perangkat Lunak (RPL)** — a Software Engineering program at a Vocational High School (SMK). That foundation lit a spark that never went out.

Today, I thrive at the intersection of **modern technology** and **beautiful design**. I don't just write code — I craft experiences. From architecting robust backend systems to building sleek, pixel-perfect user interfaces, I bring a designer's eye and an engineer's mindset to everything I create.

- 🚀 **Growing every day** — I embrace challenges as stepping stones
- 🎨 **Visual-first thinking** — I'm passionate about modern, aesthetic UI/UX
- ⚡ **Tech enthusiast** — Always exploring the latest in the ecosystem
- 🌍 **From SMK RPL to the world** — Proof that great things start anywhere
